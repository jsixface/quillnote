Κρατήστε πανέμορφες markdown σημειώσεις όποτε έχετε έμπνευση. Τοποθετήστε τις σε τετράδια και προσθέστε τις κατάλληλες ετικέτες.
Μείνετε οργανωμένοι φτιάχνοντας λίστες και θέτοντας υπενθυμίσεις.

Το Quillnote είναι 100% δωρεάν και λογισμικό ανοιχτού κώδικα (ΕΛΛΑΚ).
Δεν θα σας προβάλλει ποτέ διαφημίσεις, δεν θα σας ζητήσει πότε άδειες που δεν χρειάζεται και δεν θα ανεβάσει πουθενά τις σημειώσεις σας χωρίς την δική σας συγκατάθεση.

Με το Quillnote, μπορείτε να:

- Κρατήσετε σημειώσεις με υποστήριξη Markdown
- Φτιάξετε λίστες
- Καρφιτσώσετε τις αγαπημένες σας σημειώσεις
- Κρύψετε σημειώσεις που δεν θέλετε να δουν άλλοι
- Θέσετε υπενθυμίσεις για γεγονότα που δεν θέλετε να χάσετε
- Ηχογραφήσετε την φωνή σας και να επισυνάψετε άλλα αρχεία
- Ομαδοποιήσετε σημειώσεις σε τετράδια
- Προσθέσετε ετικέτες σε σημειώσεις
- Αρχειοθετήσετε σημειώσεις
- Αναζητήσετε σημειώσεις
- Συγχρονίσετε τις σημειώσεις σας με το Nextcloud (πειραματικό στάδιο)
- Κρατήσετε αντίγραφα ασφαλείας που μπορείτε να ανακτήσετε αργότερα
- Διαλέξετε μεταξύ dark & light mode
- Διαλέξετε το χρώμα της αρεσκείας σας για τις σημειώσεις σας ή ακόμα και για την ίδια την εφαρμογή.
